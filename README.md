# study_graph

Учебный проект для отработки технологий и изучения алгоритмов.

Возможности:

- Графическое редактирование графов
- Наблюдение за работой базовых алгоритмов с графами
- Записи работы алгоритмов с пошаговым выполнением

Особенности реализации:

- Клиент-серверное решение
- Сервер - C++ и userver
- Клиент - QML
- СУБД - Postgresql
- Локальный запуск в docker compose или в Minikube

Что планируется изучить в процессе реализации проекта:

- Разработка backend на C++ и userver
- Разработка frontend на QML
- Применение подхода DDD
- Применение SQL на практике
- Изучение алгоритмов для работы с графами
